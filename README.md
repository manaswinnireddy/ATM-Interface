# ATM Interface System

A console-based ATM Interface application developed using Java.  
This project simulates the core functionalities of an Automated Teller Machine (ATM) with secure user authentication and banking operations.

## Features

- User Login Authentication using User ID and PIN
- Multiple User Accounts
- Deposit Money
- Withdraw Money
- Transfer Money Between Users
- Transaction History with Date & Time
- Balance Inquiry
- Persistent Data Storage using File Handling
- Real-time Balance Updates in `users.txt`
- Self-transfer Prevention
- Invalid User Validation
- Console-based Interactive Menu

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- File Handling
- Arrays & ArrayLists
- Scanner Class

## How to Run

### Compile the Project

```bash
javac *.java
```

### Run the Project

```bash
java Main
```
